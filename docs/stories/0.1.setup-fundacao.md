## Story
**As a** lojista (dono do Forja3D),
**I want** um projeto Next.js com CI e deploy contínuo funcionando desde o dia 1,
**so that** toda story seguinte possa ser entregue e testada em produção imediatamente.

## Acceptance Criteria
1. App Next.js (App Router, TypeScript) criado em `app/` (ou raiz do repo) com React Three Fiber + drei instalados e página inicial renderizando um cubo 3D de smoke test
2. `npm run lint`, `npm run typecheck`, `npm run test`, `npm run build` existem e passam
3. GitHub Actions roda lint + typecheck + test + build em cada push/PR
4. Deploy automático na Vercel: preview por PR, produção na `main`
5. Storage (Cloudflare R2 ou S3) provisionado com bucket `forja3d-assets` (modelos GLB + snapshots) e credenciais via env
6. `.env.example` documenta todas as variáveis (DB, storage, Stripe, Resend — placeholders)
7. Decisões de stack registradas pelo @architect em `docs/architecture/decisoes.md` (ORM, provedor de Postgres, provedor de storage)
